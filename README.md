# 📄 Analisador de Currículos

## 🚀 Sobre o projeto
O Analisador de Currículos é uma aplicação desenvolvida para automatizar a leitura e análise de currículos.

A ferramenta extrai informações relevantes dos arquivos enviados e utiliza inteligência artificial para gerar uma análise estruturada, facilitando a compreensão do perfil profissional.

---

## 🎯 Objetivo
Facilitar a avaliação de currículos, tornando o processo mais rápido, organizado e eficiente, tanto para candidatos quanto para recrutadores.

---

## 🎓 Projeto Acadêmico
Este projeto foi desenvolvido como trabalho acadêmico do curso de Análise e Desenvolvimento de Sistemas (ADS) da Cesuca, com o objetivo de aplicar conceitos de programação, análise de dados e uso de inteligência artificial.

## ⚙️ Funcionalidades

  - Desenvolvimento de Software
  - Segurança da Informação
  - Ciência/Análise de Dados
  - Cloud Computing
  - Gestão de Projetos
  - Suporte Técnico
- Análise com inteligência artificial
- Recomendações profissionais
- Botão "Ler mais" para descrições longas
